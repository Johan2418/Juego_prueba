# Proyecto Unity 2D Pixel Art

Demo inicial de videojuego 2D creada en Unity.

## Objetivo

Construir una demo jugable de 3 a 5 minutos con estilo pixel art.

## Version de Unity

Unity 6000.0.76f1

Ejemplo:

```txt
Unity 6.x LTS o Unity 2022.3 LTS
```

## Paquetes recomendados

- 2D Sprite
- 2D Pixel Perfect
- Cinemachine
- Input System
- Unity Test Framework
- Aseprite Importer, si se usa Aseprite
- 2D Tilemap Extras, si se usan Tilemaps


## Como trabajar con Codex

Antes de pedir codigo:
1. Explicar la mecanica deseada.
2. Indicar si el juego sera lateral o vista superior.
3. Pedir cambios pequenos.
4. Revisar el resumen de archivos modificados.
5. Probar en Unity antes de continuar.
