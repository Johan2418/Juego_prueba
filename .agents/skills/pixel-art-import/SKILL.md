# Pixel Art Import Skill

Use this skill for pixel art asset setup.

## Rules

- Preserve crisp pixel art.
- Recommend Point filtering when appropriate.
- Avoid compression for important sprites.
- Keep Pixels Per Unit consistent.
- Separate characters, tilesets, UI and VFX.
- Explain slicing for spritesheets.
- Mention Pixel Perfect Camera when relevant.

## Checkpoints

Before finalizing, mention:
- Import settings.
- Pixels Per Unit.
- Filter mode.
- Compression.
- Sprite mode.
- Animation setup, if applicable.
